## 🚀 Mission Statement

**ALL-IN-One IPTV** redefines how digital broadcasting lists are aggregated, healed, and distributed. We built a system that actively repairs broken streams using a proprietary AI Quantum Healer, aggregates hundreds of thousands of channels in milliseconds, and delivers them across ultra-premium frontends (Web, Android, Desktop) without breaking a sweat.

## ⚡ Quick Start

### Python Engine (CLI)
```bash
# Require Python 3.12+
pip install -r engine/requirements.txt
python3 -m engine.src.main
```

### Docker Compose
```bash
./scripts/run_local.sh
# Access the web dashboard at http://localhost:8080
```

### Android & Desktop Apps
```bash
./scripts/build_local.sh
```

## 🧩 Deep Feature Breakdown

- **Python Engine (`engine/`)**: The core brain. Written in Python 3.12+ using `asyncio` and `aiohttp`. Capable of processing 1M+ lines per minute.
- **AI Quantum Healer (`engine/src/ai_healer.py`)**: Uses heuristic algorithms to score streams based on latency, resolution, and historical uptime ($QRS = w_1 S + w_2 (1 - L/L_{max}) + w_3 P$).
- **Android Ktor Gateway (`apps/app_proxy/`)**: A Kotlin-based microservice that translates standard M3U files into a full Xtream Codes API emulation layer (`/player_api.php`).
- **Flutter App (`apps/app_player/`)**: Glassmorphic UI with cross-platform desktop/mobile support.
- **Web Player (`player/web/` & `index.html`)**: HLS.js video streaming player with EPG timeline and search filters.
