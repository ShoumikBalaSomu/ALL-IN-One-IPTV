## 🏗️ Architecture & Component Overview

```

├── apps/

│   ├── src/                # Scraper, Verifier, Deduplicator, EPG, Search, Encryption, IPFS
```

## ⚡ Quick Start & Usage Guide

### 1. Download Production Applications
- 📱 **Flutter Unified Media Player**: [app_player-release.apk](https://github.com/ShoumikBalaSomu/ALL-IN-One-IPTV/raw/main/app_player-release.apk)
- ⚡ **Android Local Proxy Service**: [app_proxy-release.apk](https://github.com/ShoumikBalaSomu/ALL-IN-One-IPTV/raw/main/app_proxy-release.apk)

### 2. Live M3U Playlist Feeds (Auto-Updated Every 6 Hours)
- 🟢 **100% Verified Alive Feed**: 
  `https://raw.githubusercontent.com/ShoumikBalaSomu/ALL-IN-One-IPTV/main/output/checked_combined_by_country.m3u`
- 🌐 **Master Combined Feed**: 
  `https://raw.githubusercontent.com/ShoumikBalaSomu/ALL-IN-One-IPTV/main/output/combined_by_country.m3u`

### 3. Run Backend Engine Locally
```bash
# Clone the repository
git clone https://github.com/ShoumikBalaSomu/ALL-IN-One-IPTV.git
cd ALL-IN-One-IPTV

# Run the python aggregator engine
python3 -m engine.src.main

# Run unit tests
python3 -m unittest discover -s engine/tests
```

### 4. Build Applications Locally
```bash
# Build all local targets
./build_local.sh
```
