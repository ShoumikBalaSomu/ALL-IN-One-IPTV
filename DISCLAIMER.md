## 1. What This Project Is

ALL-IN-One-IPTV is an **open-source software ecosystem** that:

1. **Collects** publicly available M3U playlist URLs from open-source GitHub repositories and public websites
2. **Combines** multiple playlists into unified, organized formats
3. **Deduplicates** overlapping channel entries
4. **Verifies** that linked URLs are accessible
5. **Provides** player applications to consume M3U playlists

**All playlist URLs in this repository are sourced from publicly available, open-source projects.**

## 6. DMCA & Copyright Claims

Because we **do not host any content**, DMCA takedown requests directed at this repository are **technically ineffective** for removing the actual streams.

**For copyright owners:**
1. A URL in our text file ≠ hosting your content
2. Removing the URL from our repo ≠ removing the stream from the internet
3. Direct takedown notices to the **actual hosting providers**
4. You can identify hosts by performing **WHOIS lookups** on the stream URLs

However, if you believe a specific URL in our aggregator violates your rights, you can:
- Open an Issue requesting its removal from our source list
- Email the maintainers (see repository)
- We will review and remove confirmed sources promptly
