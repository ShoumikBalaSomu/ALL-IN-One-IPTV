Getting Started

Welcome to the ALL-IN-ONE IPTV documentation. Here you'll find everything you need to set up and use the platform.

Installation

git clone https://github.com/ShoumikBalaSomu/ALL-IN-One-IPTV.git
cd ALL-IN-One-IPTV
npm install

API Reference

Access our free IPTV endpoints directly.

Get All Channels

GET /api/channels
Response:
{
  "status": "success",
  "data": [
    { "id": "1", "name": "News 24", "url": "..." }
  ]
}

Player Setup

Embed the Web Player into your own application.

<iframe src="/player/web/index.html" width="100%" height="600px"></iframe>
